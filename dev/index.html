DatadogClient.Datadog — Method
Datadog(ip::String="127.0.0.1", port=8125)
Datadog(ip::IPAddr, port)

Create a Datadog client sending metrics to the given ip and port. Should be created only once and used for all metrics.

Examples

julia> client = Datadog()
julia> increment(client, "visits", "test:true", "env:prod")
